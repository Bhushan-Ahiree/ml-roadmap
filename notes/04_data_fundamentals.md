# Data Fundamentals

## What is Data?

Data is a collection of facts or information that can be processed to solve a problem or make predictions.

Example:

Age = 24
Salary = 50000
City = Pune

A collection of this information is called a dataset.

---

## Dataset

A dataset is a collection of rows and columns.

Example

| Area | Bedrooms | Age | Price |
|------|----------|-----|-------|
|1200  |2         |5    |75     |
|1500  |3         |2    |95     |
|900   |2         |10   |52     |

---

## Row (Sample)

A single record in a dataset.

Also called:

- Sample
- Observation
- Instance
- Record

Example

1200 | 2 | 5 | 75

---

## Column (Feature)

Each column represents one property of the data.

Also called:

- Feature
- Variable
- Attribute

Examples

- Area
- Bedrooms
- Age

---

## Target 

The value we want the model to predict.

Example

Price

---

## Features

The input variables used to predict the target.

Example

Area

Bedrooms

Age

---

## Feature → Target

Area
Bedrooms
Age
        ↓
     Machine Learning Model
        ↓
      Price

---

## Key Takeaways

- Data is information.
- A dataset consists of rows and columns.
- Rows are called samples or observations.
- Columns are called features.
- The output is called the target.
