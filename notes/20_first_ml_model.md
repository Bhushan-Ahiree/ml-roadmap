# First Machine Learning Model

## What is Scikit-learn?

Scikit-learn is the most popular Python library for traditional Machine Learning.

It provides ready-to-use implementations of ML algorithms.

---

## Regression

Regression predicts continuous numerical values.

Examples

- House Price
- Salary
- Temperature

---

## Linear Regression

Linear Regression tries to find the best straight-line relationship between input features and the target.

Example

Area → House Price

As area increases,

Price usually increases.

---

## Features (X)

Input variables used for prediction.

Example

Area

Bedrooms

Age

---

## Target (y)

The value we want to predict.

Example

House Price
