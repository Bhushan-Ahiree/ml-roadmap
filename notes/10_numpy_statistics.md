# NumPy Statistics

## Sum

```python
arr.sum()
```

Returns the total of all elements.

---

## Mean

```python
arr.mean()
```

Returns the average.

---

## Maximum

```python
arr.max()
```

Returns the largest value.

---

## Minimum

```python
arr.min()
```

Returns the smallest value.

---

## Standard Deviation

```python
arr.std()
```

Measures how spread out the values are.

A low standard deviation means values are close to the average.

A high standard deviation means values are more spread out.

---

## Key Takeaways

sum() → Total

mean() → Average

max() → Largest value

min() → Smallest value

std() → Spread of data
