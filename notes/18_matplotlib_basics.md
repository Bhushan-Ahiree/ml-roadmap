# Matplotlib Basics

## What is Matplotlib?

Matplotlib is the most popular Python library for data visualization.

It helps us understand data by creating graphs.

---

## Why Visualization?

Visualization helps us:

- Find patterns
- Detect outliers
- Understand distributions
- Compare categories
- Explain results

---

## Common Charts

Line Chart

Used for trends over time.

Bar Chart

Used to compare categories.

Histogram

Used to understand data distribution.

Scatter Plot

Used to study relationships between variables.

---

## Basic Syntax

```python
import matplotlib.pyplot as plt
```

Show graph

```python
plt.show()
```

---

## Key Takeaways

Visualization is an important part of EDA.

Always understand your data before training a model.
