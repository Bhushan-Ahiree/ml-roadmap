# Pandas Introduction

## What is Pandas?

Pandas is a Python library used for working with structured data.

It allows us to load, clean, analyze, and manipulate data efficiently.

---

## Why Pandas?

Machine Learning datasets are usually stored as:

- CSV
- Excel
- SQL
- JSON

Pandas makes working with these datasets easy.

---

## Difference between NumPy and Pandas

NumPy

- Numerical computing
- ndarray
- Faster mathematical operations

Pandas

- Data analysis
- Series
- DataFrame
- Works with tables

---

## Series

A Series is a one-dimensional labeled array.

Example

Age

20

22

24

26

---

## DataFrame

A DataFrame is a two-dimensional table.

Example

| Name  | Age |
|------ |---- |
| John  | 22  |
| Alice | 24  |

---

## Key Takeaways

NumPy → Arrays

Pandas → Tables

Machine Learning datasets are usually handled using Pandas.
